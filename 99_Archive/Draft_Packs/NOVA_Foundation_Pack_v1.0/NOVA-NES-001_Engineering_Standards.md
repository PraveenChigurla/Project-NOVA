
# NOVA-NES-001 — Engineering Standards

Rules:
1. Architecture before implementation.
2. One responsibility per module.
3. Strong typing.
4. Comprehensive logging.
5. Configuration outside code.
6. Unit tests required.
7. Clear documentation.
8. Dependency injection where appropriate.
9. Avoid circular dependencies.
10. Security-first design.
11. Explainable behavior.
12. No hidden side effects.

Every implementation must include:
- README/SPECIFICATION
- Tests
- Error handling
- Logging
