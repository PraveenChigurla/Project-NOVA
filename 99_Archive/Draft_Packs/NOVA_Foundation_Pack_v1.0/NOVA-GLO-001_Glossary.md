
# NOVA Glossary

Capability: Major reusable platform ability.
Skill: Combination of capabilities.
Engine: Major subsystem.
Service: Backend implementation.
Tool: Executable utility/library.
Workflow: Ordered execution of capabilities.
Environment: Desktop, Browser, Terminal, etc.
Provider: External implementation (LLM/OCR/etc.).
Adapter: Interface between NOVA and external systems.
Action: Lowest-level executable operation.
