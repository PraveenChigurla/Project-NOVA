# 05_Templates

This directory holds the official templates for all Product, Architecture, Engineering, and Design documents.

## Contents

- `PRS_Template.md` - Product Requirements Specification template.
- `SRS_Template.md` - Software Requirements Specification template.
- `HLD_Template.md` - High Level Design template.
- `LLD_Template.md` - Low Level Design template.
- `ADR_Template.md` - Architectural Decision Record template.
- `NES_Template.md` - NOVA Engineering Standard template.
- `Capability_Spec_Template.md` - Capability Specification template.

## Purpose

To ensure consistent structure, quality, and meta-information across all documentation generated by team members and agents.
