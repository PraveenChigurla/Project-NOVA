# NOVA Product Backlog

Unlike infrastructure backlogs organized by technical subsystems, the NOVA Product Backlog is organized entirely by human intentions. Every completed experience represents a fully integrated, seamless product feature where the AI Kernel, Planner, and Providers work invisibly to accomplish what the user says.

## The North Star KPI
Every completed goal increments the system heartbeat:
- Commands Executed
- Mouse Clicks Saved
- Keyboard Presses Saved
- Time Saved
- Applications Controlled
- Successful Goals
- Skipped Actions

> **Every Experience must be expressed first as a sentence a user naturally says. Only after the sentence is accepted should the implementation be designed.**

---

## 🚀 The 10 Experience Roadmap

### Experience 01: Prepare Workspace [✅ COMPLETE]
**"NOVA, prepare my development workspace."**
- Intelligent skipping of running processes
- Dynamic terminal launching
- Git status checks (no auto-resolves)
- Goal Reporting

### Experience 02: Universal Computer Control [NEXT]
**"Open anything. Control my system."**
- Universal Application Control (Open, Close, Switch)
- Window Management (Minimize, Maximize, Move, Resize, Focus)
- System Controls (Volume, Brightness, WiFi, Bluetooth, Sleep, Shutdown)
- Avoid duplicate launches intelligently.

### Experience 03: Screen Interaction
**"Click anything. Find anything."**
- Semantic Interaction ("Click Login", "Choose Personal")
- Scrolling via semantic intent ("Scroll until Settings")
- Target Resolution via OCR and Vision (No hardcoded coordinates)
- Spatial Reasoning ("Click Login below Password")

### Experience 04: Developer Assistant
**"Review my code. Run my tests."**
- Run tests and summarize failures
- Git automation (Status, Commit, Push, Pull)
- Explain tracebacks
- Search TODOs and read docs

### Experience 05: Browser Assistant
**"Browse for me. Read this page."**
- Natural language browser automation (DOM preferred, Vision fallback)
- Summarize pages and emails
- Download invoices
- Fill forms

### Experience 06: File Assistant
**"Organize my files."**
- Rename, Move, Compress, Delete
- Find duplicates

### Experience 07: Morning Routine
**"Start my day."**
- Automated reading of calendar, email, and task boards
- Synthesis of daily priorities

### Experience 08: Daily Review
**"Wrap up my day."**
- Summarize completed work and active branches
- Prepare context for tomorrow

### Experience 09: Meeting Assistant
**"Join my meeting and take notes."**
- Screen and audio context integration
- Automated summarization

### Experience 10: AI Copilot Everywhere
**"Always there when I need you."**
- Global hotkey, ubiquitous presence across all OS layers
