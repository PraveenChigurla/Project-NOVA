# 03_Experiments

This directory contains temporary prototypes, spike solutions, playground code, and ad-hoc scripts.

## Purpose

To provide developers with a sandbox area to explore new libraries, test code snippets, and play with drafts without polluting research or production code.

## Guidelines

- All contents in this directory should be considered throwaway.
- Files should be named with prefix dates or developer initials to prevent collision (e.g., `20260628_pg_test_db.py`).
